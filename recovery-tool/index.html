How this works:
  1. Queries Nostr relays for released shares (kind 30080)
  2. Decrypts shares using your private key (NIP-44)
  3. Reconstructs the encryption key (Shamir 3-of-5)
  4. Fetches the encrypted message (kind 30081)
  5. Decrypts the message (AES-256-GCM)
