The Problem

If something happens to you, how do your loved ones access critical information? Passwords die with you. Lawyers can be compromised. Cloud services get shut down.

The Solution

message + time lock = autonomous release

Write a message. Set a timer. Check in regularly. If you stop checking in, your message releases automatically to your chosen recipients.

How It Works

1

Encrypt

Your message is encrypted with AES-256-GCM. Military-grade. Unbreakable.

2

Shatter

The encryption key is split into 5 fragments using Shamir's Secret Sharing. Need 3 to reconstruct.

3

Scatter

Fragments distributed across 7+ independent Nostr relays worldwide. No single server to compromise.

4

Timestamp

Bitcoin blockchain anchors the timer. 800,000+ nodes verify. Unstoppable.

5

Release

Timer expires. Fragments reunite. Message decrypts. Recipients notified.

Decentralized Network

Your fragments live on independent Nostr relays run by different people in different countries. Even if 4 relays disappear, your message survives.
